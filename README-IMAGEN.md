# Imagen de Santa para Realidad Aumentada

Este directorio debe contener la imagen de Santa que se mostrará en la experiencia de Realidad Aumentada.

## Instrucciones

1. Descarga una imagen de Santa con fondo transparente (PNG recomendado)
2. Nombra el archivo como `santa.png`
3. Colócalo en esta carpeta (`/public/`)

## Características recomendadas

- **Formato**: PNG con transparencia
- **Tamaño**: Máximo 1024x1024 píxeles
- **Peso**: Menos de 500KB
- **Orientación**: Santa debe estar orientado hacia la cámara

## Ejemplos de imágenes gratuitas

Puedes encontrar imágenes de Santa con fondo transparente en estos sitios:
- [Freepik](https://www.freepik.com/free-photos-vectors/santa-claus)
- [PNG Mart](https://www.pngmart.com/santa-clip-art/)
- [Pixabay](https://pixabay.com/vectors/search/santa%20claus/)

## Nota

Si no agregas una imagen, la aplicación mostrará un marcador de posición.
